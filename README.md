# Zenova - E-commerce Zen & Naturale

Sito e-commerce con design caldo, soft e zen per prodotti naturali e benessere.

## Caratteristiche

- Design minimalista e zen con palette di colori caldi
- Catalogo prodotti con 9 prodotti iniziali
- Carrello della spesa funzionale
- Salvataggio automatico del carrello (LocalStorage)
- Design responsive
- Animazioni fluide e transizioni soft

## Come Utilizzare

1. Apri il file `index.html` nel tuo browser
2. Naviga tra i prodotti
3. Aggiungi prodotti al carrello
4. Visualizza e gestisci il carrello

## Struttura del Progetto

```
zenova-ecommerce/
├── index.html          # Pagina principale
├── styles.css          # Stili CSS
```
